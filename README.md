# Excel Dashboards 

Collection of business-focused Excel dashboards built using Pivot Tables, calculated fields, and advanced data visualization.

## Dashboards Included

### 1. E-commerce Dashboard
- Sales, Orders, Quantity, and Profit KPIs
- Profit Margin by Category (loss-making category identified)
- Category-wise Sales & Profit analysis
- Sales vs Discount impact on Quantity
- Monthly Sales & Profit trends
- Customer Demographics
- Top 10 Customers by Sales

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Calculated Fields
- Dashboard Design & Formatting
