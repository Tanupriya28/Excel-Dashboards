# E-commerce Excel Dashboard

## Overview
Interactive Excel dashboard analyzing e-commerce sales, profit, discounts, customer behavior, and profitability.

## Key Features
- KPI cards for Sales, Orders, Quantity, Profit
- Profit Margin by Category (highlighting loss-making Grocery category)
- Category-wise Sales & Profit comparison
- Sales vs Discount analysis
- Monthly Sales & Profit trends
- Customer Demographics and Top Customers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Calculated Fields
